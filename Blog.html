Progressive Web Apps

Web pages and mobile app, Sep 20 ,

PWA’s are type of application delivered using web technologies.

They can work on any system with the updated browser like windows, ios and android. PWA work like standard phone apps i.e. can work offline, interact with operating system like sending notification. The biggest advantage using this system is that user doesn’t have to install an application to use this. work on PWA was started by google in 2015. After that all major browsers started supporting it one after that other. As of now, PWA’s can work on all type of screens like phones, tablets and desktops. PWA’s provide reliability, easy installation process, speed etc. Listing few examples of PWA’s : 1. Twitter Lite 2. Pinterest 3. Flipboard 4. Flipkart 5. Telegram

Web Game Development

Play game over web pages, Sep 27,

Video games are one of the most graphically and technically challenging computing problem.

That is why most of the times they are developed using platform native technologies. Modern Web has been slowly improving to start becoming a viable platform for video game developments and allowing developers to deploy games for all type of platforms The biggest driver for this is massive improvements in Javascript performance ( thanks to JIT compiler technology) and modern WebAPI’s. There are several game engines like canvas which help develop all sort of games ranging from small games to huge RPG’s..
