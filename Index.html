Progressive Web Apps

PWA’s are type of application delivered using web technologies. they can work on any system with the updated browser like windows, ios and android. PWA work like standard phone apps i.e. can work offline, interact with operating system like sending notification . the biggest advantage using this system is that user doesn’t have to install an application to use this. work on Pwa was started by google in 2015. they where able to give concepts of service workers and web app manifests.
